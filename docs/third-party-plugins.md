| Name             | Description                              | Link                                                                                      |
|:-----------------|:-----------------------------------------|:------------------------------------------------------------------------------------------|

| proxying / stubbing imports | Fusebox proxying / stubbing imports  Plugin| [tomitrescak/proxyrequire](https://github.com/tomitrescak/proxyrequire) |
